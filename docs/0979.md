# 你对谁负责？

> 原文：<https://alphamale20.com/2017/02/20/who-are-you-responsible-for/>

我一直在想我对谁的幸福负有责任。是世界吗？这似乎是不可能的，充满了 SP 的废话。只有我自己吗？这似乎是疏远所有人的一个简单方法。你的底线在哪里？家人？朋友？那些和你做生意的人？你的员工？你当地的社区？那些有共同文化或意识形态的人？你的国家？利他和利己在什么点结束，SP 在什么点开始？什么时候你不再给予严厉的爱，而是变成了一个混蛋？蒂姆

好问题。你到底对谁负责？在我给你我的答案之前，把这个放在社会规划鼓励的其他常见答案的背景下可能更有意义。

左翼观点
如果你的世界观更加左倾，你的责任层级可能是这样的:

1.你非常亲密的家庭成员(配偶、孩子，也许还有父母)。
2。你自己。
3。你的密友和其他家庭成员。
4。整个世界。

很自然，你认为你自己和你的家庭非常重要。那是标准的人类物品。但是，你可能会认为整个世界和整个人类都是你应该负责的。宗教和国家之类的东西可能对你来说并不重要，但环境和核武器问题却是你认为极其重要的，因为这些东西威胁着“世界”。

你可能觉得对“每个人”都有责任去拯救和保护整个世界，尤其是穷人和/或那些你认为受压迫的人。目睹人们不关心这个世界或全球人类可能会让你非常不安。

右翼观点
如果你是更保守的右翼，你的观点是“传统的”，你的责任等级看起来与左翼非常不同。它可能看起来像这样:

1.上帝，耶稣，或者你的宗教信仰。
2。你深爱的人。
3。你自己。
4。你的大家庭。
5。你的国家。
6。你的种族。

像左翼分子一样，你认为你的亲密家庭成员非常重要。再说一遍，这是正常的人类行为。与左翼分子非常不同，你可能认为你的宗教、你的国家和你的种族极其重要。对你来说，你的国家(可能还有你的种族)是第一位的。这并不意味着你一定反对其他国家或种族(尽管今天许多愤怒的保守派都反对)，但这无疑意味着关心“整个世界”对你来说没有多大意义。你觉得有必要先关心你的国家/种族，然后再去关心其他国家/种族(或者根本不关心)。

在看到人们不关心“世界”让左翼人士不爽的同时，看到人们不关心你的国家让你不爽。

安·兰德的观点现在让我们来看看安·兰德和像她一样的客观主义者所支持的这两种社会观点的截然相反的观点。在她看来，你的责任层级如下:

1.你自己。
就是这样！她的观点是 100%自私，不在乎任何人，除非你选择这样做，即使这样，只有当那些人值得这样的关注。显然，这种信念公然对抗左翼和右翼的社会规划，因此不太受欢迎，也永远不会受欢迎。

正如你可能想象的那样，我的观点既不左也不右。这有点接近安·兰德的观点，尽管它远没有那么极端。

我的责任层次结构如下所示:

1.你自己。
2。你的核心圈子，定义为你的孩子(如果有的话)，你的配偶或同等的人(如果有的话)，可能还有你的父母。
3。物有所值的关系。
我会解释的。

我的首要责任是对我自己。不是我的家庭，不是我的宗教，不是这个世界，不是我的国家，不是我的种族。如果我的生活很糟糕，这些对我来说都不重要。因此，我的第一要务是我自己，确保我的生活不会糟糕。因为我生活中的一切都是我的错，所以除了我自己，绝对没有人能让我的生活变得精彩。我是说这个。我的女朋友，我的爸爸，我的朋友，我的孩子，我的顾客，唐纳德·川普，耶稣，伯尼·桑德斯，米洛·伊安诺普洛斯，这些人没有一个会让我的生活变得精彩或解决我的问题。一个都没有！一个都没有！这些人大多数都不在乎，极少数在乎的人有他们自己的问题要解决，因此他们没有时间让我的生活变得精彩。

因此，由我决定。这就是为什么他们在航空安全演讲中告诉你，在你开始帮助别人戴上氧气面罩之前，先把氧气面罩戴在自己身上。如果我没有钱，没有性生活，或者身体很差，我就不能帮助这个世界(或者我的家庭，或者我的国家)。从我开始。我是我生命中第一个需要解决的人或实体。所有人和所有事都是第二位的。他们必须这么做。

一旦我知道我被照顾，现在我可以帮助别人。我可以处理下一个优先名单上的人，那些我亲近的家庭成员。“亲密”家庭成员是指我选择拥有的家庭成员。那将是我的孩子(如果我有孩子的话，就我而言，我有)和我的配偶或类似的人(同样，如果我有孩子的话，我现在差不多有了)。在这两种情况下，你都选择了要孩子(我希望如此),也选择了要配偶。没人逼你；你选择了这些人。这不像你的父母、兄弟姐妹、阿姨、叔叔等等。这些人是强加给你的；你没有选择拥有它们。因此，你不需要对这些人负责，除非你选择这样做。

以我来说，我选择对父母负责一点。我没有选择对我的大家庭中的任何其他人负责，我也不会。我有许多兄弟姐妹，我爱他们所有人，但他们的生活是他们自己的问题。我已经忙着管理我的了。我个人选择了我的父母，因为与许多人不同，我有一个美好的童年，并且有两个非常好的父母。如果你有差劲的父母，我认为选择对他们负责是错误的。尊重和责任是赢得的。如果你父母很差劲，那也不是他们应得的。

我不在乎他们是否“生了你”和“养了你”他们“拥有你”是因为他们想要拥有你。正如我以前说过的，生孩子是你能采取的最自私的行为之一。(这包括我有两个孩子的时候。)你没有孩子是因为你想帮助这个世界。你生孩子是因为你出于自私的原因想要孩子。这包括你父母生下你的时候。

就你的孩子而言，我认为你应该对你的孩子 100%负责，直到他们成年，在大多数社会里，他们成年是 18 岁。我的两个孩子现在都已经 18 岁或 18 岁以上了，所以从传统意义上来说，我不再对他们负责，但当他们不满 18 岁时，我对他们非常负责。我自私地把他们带进了现实，所以由我(和他们的母亲)来照顾他们。

仅仅因为我的孩子超过 18 岁，并不意味着我现在会把他们扔进狼群，对他们漠不关心。当然，我非常爱他们，关心他们的未来。我把离开这个崩溃的国家的时间推迟到(可能)2025 年的最大原因之一是，我想确保我的成年子女在我离开之前都已经成年。所以很明显我在乎。只是我不再对他们负责了。他们现在是成年人了。他们的责任是对他们自己。

当我的女朋友粉红萤火虫搬来和我一起住的时候，我会觉得对她有点责任。我说了一点。她不是一个孩子；她是一个非常能干和独立的成年人；这就是我爱她的原因。所以我不打算为她负全责；那将是愚蠢的。但就我要对谁负责而言，她肯定是名单上的第二位，就在第一位:我之下。

所以，如果你想更清楚地想象我的清单，那应该是我，当她搬进来时，我会变成部分粉红色，然后是我的孩子，然后是我的父母，但前提是他们长大后需要我的帮助。他们可能不会，或者我的兄弟姐妹可能会站出来提供这种护理。哪种方式对我来说都可以。

这涵盖了我所负责的人的核心圈子。我对任何人都不负责。社会、我的种族、我的国家、耶稣、世界，这些东西对我来说都不重要，因为 A)它们中的大多数都是自我维持、自我纠正的系统，B)它们中的许多都以非理性、自杀的方式行事，憎恨像我这样的人，并且不需要我的帮助，正如我在这里解释的那样。
世界？不管你我做什么或不做什么，它都会继续下去。甚至威胁人类的世界范围的环境危机也不会发生，因为人类会在最后一刻发明一些东西来解决问题。他们总是这样。

我的国家？如果美国失败了(它最终会失败的)，没问题，我会在别的地方，很远的地方。如果那个国家失败了，没问题。作为一个阿尔法男性 2.0，脱离系统，有位置独立的收入，我就搬到别的地方。世界上总会有正常运转的国家。我的种族？嗯，我是白人，现在大多数白人在涉及到他们种族的未来时都有自杀倾向，所以我早就不在乎这个了。如果我的比赛失败了(它会；白人就要拜拜了)，我才不管呢。不管怎样，今天的亚洲人更聪明、更勤劳，混血人也很有吸引力。见鬼，我已经创造了至少一个白人(我的女儿)，所以我对我的种族的任何义务在很久以前就已经完成了。
宗教？西方宗教已经失败了，所以我不需要担心这个，特别是过着阿尔法 2.0 的生活方式，位置独立，金融多样化和非单一性。

如果我对这些事情都不负责，那么在我的核心圈子之外，我对什么负责呢？难道我只是一个自私的混蛋，无所事事，坐享其成，却不给予任何回报？一点也不。我对我自愿创造的许多物有所值的关系负责。价值对价值的关系-
社会规划告诉你，你要对某些实体(家庭、世界、上帝、你的国家、你的种族等)负责，因为那是“对社会最好的东西”忘了你吧，社会更重要，你这个不重要的人。听起来不错。我在哪里报名？

当然，这预先假定你确切地知道什么对社会是最好的，而你并不知道。我也不知道。我们只是猜测什么是最好的，主要基于主观因素，如社会规划、我们自己的个性和我们自己的现状。

一种更聪明的生活方式，一条更有利于长期幸福的道路，是忽略所有主观的、情绪的、哲学的废话，专注于你创造的基于价值与价值的个人关系。物有所值是指我自愿给你你看重的东西，作为交换，你也自愿给我我看重的东西。义务与此无关。社会规划(通常)与此无关。你有我想要的东西，我有你想要的东西，所以我们自愿交易。没有人强迫我们。物有所值。
我唯一的责任就是给你提供你承诺的价值。你唯一的责任就是给我提供你承诺的价值。就是这样。不用担心这个世界，这个国家，耶稣，环境，白人，黑人，或者其他任何事情。

一个简单的例子是，如果你想学习如何更容易地上床，你买了我的一本约会书籍，我们就进行一个简短的价值对价值的关系，你学习如何上床，我得到大约 40 美元。我个人认为你得到了更好的结果，因为在我 23 岁的时候，我会不惜花费 40 或 50 美元来学习一个行之有效的系统，告诉我如何放松。但这没关系，因为我有能力参与许多这样的关系，而不仅仅是一两个。我一天能卖出很多书，每天都是，而且确实如此。所以你上床了，我实现了我的财务目标。赢/赢。物有所值。

如果出于某种原因，我对你撒谎，而我的书没有提供价值，那也没关系，因为我的书有 100%的终身保证。无论你拥有它多长时间，你都可以在任何时候归还它们，拿回你的钱。据我所知，我是 PUA 社区或 manosphere 中唯一这样做的人。这是因为，不像这个领域的其他大师，我关注的是物有所值。我想确保你得到了价值；那是除了我自己和我的核心圈子之外，我在生活中唯一的责任。

物有所值不仅仅适用于商业交易。如果你有一个 FB，你给她她想要的价值(性，也许还有关注)，她给你你想要的价值(性)。如果你有一个你喜欢与之共度时光的朋友，你给他一些有价值的东西(友谊、关注、支持、社会交往)，他也会给你同样的回报。物有所值。

你可以争辩说，有时你会得到更糟糕的结果，就像 FB 为了几分钟的性爱而占用了你很多时间。这种情况有时会发生，但是记住，你生活中的一切都是你的错，所以你有能力和那个婊子做下一步，随时随地和别人做爱(我已经这样做了)。你也可以解雇有问题的客户(我就这么干过)，甩了有毒的朋友(我就这么干过)，离开让你不开心的工作(我就这么干过)，甚至软的或硬的把你当屎的下一个家人(我就这么干过)。这是因为我只做物有所值的关系。我不允许其他类型的关系进入我的生活。我的生命和我的幸福太重要了。物有所值意味着我也得到了价值，不仅仅是你。如果你得到了价值，而我没有，我会把你从我的生活中赶出去，重新快乐起来，给别人提供价值，同时得到回报。当你的生活以价值为基础时，突然之间，你真的不在乎社会、你的国家或任何其他政治事物了。你忙着生产和快乐。你在让别人幸福的同时，自己也过着幸福的生活。这是一笔不错的交易。

更好的是，当你关注价值时，你实际上让世界变得更好。让世界变得更糟的人是那些专注于吸取价值但不回报任何价值的人，比如整天玩视频游戏而不是工作的人，不断与失败者生孩子的单身母亲，或者敲诈客户但仍不断获得纳税人资金救助的百万富翁对冲基金经理。通过关注物有所值，你实际上改善了经济和你周围人的生活，至少是一点点。你是解决方案的一部分，而不是问题的一部分，即使你实际上并不关心社会。

这就是为什么，尽管我对社会毫不在乎，但我实际上对社会的帮助要比许多人指责我因为对社会不感兴趣而伤害社会多得多。我专注于创造大规模的经济活动，并为他人带来幸福，而其他许多人专注于投票给专制政客，或在互联网上游说与他们有政治分歧的人。我们中谁对社会的帮助更大？

因此，在我看来，你的首要责任是对你自己，其次是你核心圈子里的人(孩子、配偶，也许还有父母)，然后是你创造的许多大大小小的物有所值的关系。我意识到我所说的与你们大多数人头脑中的社会规划大相径庭，我希望在左翼和右翼的评论中都能听到巨大的分歧。不管怎样，如果你专注于这三件事，而忽略其他一切，你将永远快乐，永远富有成效，即使你不在乎，也能帮助社会，成为一个好人。
